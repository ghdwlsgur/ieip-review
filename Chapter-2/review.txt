다음은 C 언어의 일부분이다. 실행 결과는 ?
char cc;
int dd = 50;
switch(dd) {
   case 50:
      cc = 'x';
   case 30:
      cc = 'y';
   default:
      cc = 'z';   
}
printf("%c", cc);

1. x
2. y
3. z
4. error 발생

정답 3. z
break문이 없으므로 case문 전체가 실행됩니다. 그러므로 cc에는 가장 마지막 실행문에 의해 'z'가 저장됩니다.

다음 C 프로그램의 실행 결과는 ?
#include <stdio.h>
main() {

   int a = 10;
   if (a == 10)
      printf("a는");
      printf("%d입니다", a);
   else
      printf("a는);
      printf("%d이 아닙니다.", a);
}

1. a는 10입니다.
2. error 발생
3. a는 10이 아닙니다.
4. 10입니다.

정답 2. error 발생
if문에서 조건이 참이거나 거짓일 때 실행할 문장이 두 문장 이상이면 중괄호({})를 입력하고
그 사이에 실행할 문장을 입력해야 하는데 중괄호({})없이 두 문장을 입력했으므로 error가 발생

다음 C 프로그램을 실행했을 경우 출력되는 값은 얼마인가 ?
main() {
   int a = 100;
   if(a = 200)
      a = 300;
   else
      a = 400;

   printf("%d\n", a);
}

1. 100
2. 200
3. 300
4. 400

정답 3. 300
"if(a = 200)"처럼 대입연산자 =을 이용하면 a에 200이 저장되고, 200은 참이므로 'a = 300'을 수행

다음 C언어 프로그램이 실행되었을 때의 결과는 ?
#include <stdio.h>
int main(void) {
   int n = 4;
   int *pt = NULL;
   pt = &n;

   printf("%d", &n + *pt - *&pt + n);   
   return 0;
}

1. 0
2. 4
3. 8
4. 12

정답 3. 8
&n => n의 주소 
*pt => n의 주소값 = 4
*&pt => n의 주소는
n => 4
